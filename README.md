# Music Library

---

## Requisitos
- Node.js LTS ≥ 22
- npm v10+
- npx v10+ (incluido con npm moderno)

---

## Instalación

1. **Clonar el repositorio**
```bash
git clone <repo-url>
cd music-library
```
2. **Instalar dependencias**
```bash
 npm install
```
3. **Ejecutar el servidor de desarrollo**
```bash
 npm run dev
```
> Por defecto, la aplicación se sirve en `http://localhost:8080`. Si el puerto está ocupado, revisa `vite.config.ts`.
